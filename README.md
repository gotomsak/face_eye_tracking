# face_eye_tracking

## 集中度の公式
* C = wc1 + (1-w)c2
* c1: 正規化した瞬き回数を頻度に変換したもの
* c2: 正規化した動きの変化量を頻度に変換したもの
* w:  正面からの動き

## 正規化の式
* (list[i] - max(list)) / (min(list) - max(list)
## wの式
* w = 0 (r = 0, p = 0, y = 0)
* w = 1 - ((r/tr + p/tp + y/ty) / 3)
* r: roll
* p: pitch
* y: yaw
* tn: nの閾値
* r,p,yは動画の1フレーム目でキャリブレーション
