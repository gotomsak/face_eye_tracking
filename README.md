# face_eye_tracking

## 集中度の公式
* C = c3c1 + (1-c3)c2
* c1: 正規化した瞬き回数
* c2: 正規化した動きのフレーム数
* c3: 正規化したよそ見したときのフレーム数

## 正規化の式
* (list[i] - max(list)) / (min(list) - max(list)
